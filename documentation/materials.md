# Materials

- [Binder's thread](https://www.modulor.de/en/linen-binder-s-thread-th-0-42mm-type-30-3-150m-black.html)
- PVA glue
- [Bookmark ribbon](https://www.schmedt.de/zeichenlitze-lesezeichenband-spitzenqualitaet/50825483)
